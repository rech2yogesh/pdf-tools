How it works

AES-GCM 256-bit — military-grade encryption via the browser's built-in Web Crypto API.

Encrypted files are saved as .enc — only this tool (or compatible AES-GCM tools) can decrypt them.

The password is never stored anywhere. Lose it and the file cannot be recovered.

Works with any file type — PDF, image, spreadsheet, archive, anything.

Tips

What gets removed — Author name, creation date, modification date, software name, and company field.

Why it matters — Recipients can view metadata in Adobe Acrobat or Chrome — strip it before external sharing.

Re-rendered — The cleaned PDF is fully re-rendered, not just patched, ensuring no hidden data remains.

Privacy by design. Every operation runs locally in your browser using open-source libraries — no file is ever uploaded to a server, and everything is cleared from memory when you close or refresh this tab.
